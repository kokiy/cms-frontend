# 项目概述

## 1.1 项目简介

这是一个现代化的博客内容管理系统前端项目，采用当前主流的 React 技术栈构建。

## 1.2 本期交付范围

| 功能模块     | 说明                                                      |
| :----------- | :-------------------------------------------------------- |
| **登录认证** | 账号密码登录、注册、登录态持久化、401 失效处理            |
| **文章管理** | 列表查询、分页、关键字搜索、状态筛选、创建/编辑、发布配置 |
| **标签管理** | 标签的新增、编辑、删除、文章关联计数展示                  |
| **全局布局** | 侧边栏、顶栏面包屑、内容区布局一致性                      |
| **国际化**   | 使用 react-intl 实现中英文切换                            |

## 1.3 本期不做

- 多角色权限系统（仅保留"已登录可访问"最小权限模型）
- 文章版本历史、协同编辑、回收站
- 多主题品牌化

## 1.4 用户角色

| 角色   | 权限范围                 |
| :----- | :----------------------- |
| 管理员 | 可访问全部 v1 页面与操作 |
