# 项目架构

## 4.1 整体架构

```
┌─────────────────────────────────────────┐
│           页面层 (Pages)                 │
│  LoginPage, PostsPage, TagsPage         │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│        组件层 (Components)               │
│     Layout, UI Components                │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│    路由与状态管理层                       │
│  (Router, Providers, Stores)            │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│      服务层 (Services)                   │
│  API Client, Authentication              │
└─────────────────────────────────────────┘
```

## 4.2 状态分层

| 状态类型   | 管理工具       | 适用场景                      |
| :--------- | :------------- | :---------------------------- |
| 服务端状态 | TanStack Query | 来自 API 的数据、列表、详情   |
| 客户端状态 | Zustand        | UI 本地状态、用户偏好、临时态 |

```
src/

├── components/          # 组件目录
│   └── layout/          # 布局组件

│   │   ├── login.ts     # 登录页
│   │   ├── posts.ts     # 文章管理
│   │   └── tags.ts      # 标签管理
│   └── en-US/
├── pages/               # 页面组件
│   ├── login/
│   ├── posts/
│   └── tags/
├── providers/           # React Context Providers
├── router/              # 路由配置
├── services/            # API 服务层
├── stores/              # Zustand 状态管理
│   └── slices/

├── utils/               # 工具函数

```

## 4.4 路由配置

| 路径        | 页面说明 | 访问控制 |
| :---------- | :------- | :------- |
| `/login`    | 登录页   | 公开     |
| `/register` | 注册页   | 公开     |
| `/posts`    | 文章管理 | 需登录   |
| `/tags`     | 标签管理 | 需登录   |

> **规则**：未登录访问受保护路由时，统一跳转至 `/login`

## 4.5 Zustand Stores

| Store 名称       | 职责范围                           | 持久化 |
| :--------------- | :--------------------------------- | :----- |
| `useAuthStore`   | Token、用户信息、登录态恢复        | 是     |
| `useGlobalStore` | 侧边栏折叠、语言偏好、全局 UI 偏好 | 是     |
| `usePostStore`   | 文章编辑草稿态（未保存变更）       | 否     |
| `useTagStore`    | 标签编辑临时态                     | 否     |

## 4.6 国际化

| 语言     | Locale  | 说明 |
| :------- | :------ | :--- |
| 简体中文 | `zh-CN` | 默认 |
| 英文     | `en-US` |      |

**使用规范**：

- 使用 `useIntl()` Hook 或 `<FormattedMessage />` 组件
- **禁止**在组件中硬编码中文字符串
- 消息 Key 命名规范：`[页面].[模块].[描述]`
- 语言切换后需持久化到 `localStorage`
