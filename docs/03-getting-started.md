# 快速开始

## 3.1 安装依赖

```bash
npm install
```

## 3.2 开发模式

```bash
npm run dev
```

## 3.3 构建生产版本

```bash
npm run build
```

## 3.4 代码检查与格式化

```bash
npm run lint          # ESLint 检查
npm run lint:oxlint   # Oxlint 检查
npm run lint:style    # Stylelint 检查
npm run format        # 格式化代码
npm run gen:api       # 生成 API SDK
```
